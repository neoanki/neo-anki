# Contributing to Neo Anki

Thank you for helping build a dependable, humane spaced-repetition tool.

## Development

1. Install Node.js 24 and run `npm ci`.
2. Start the desktop app with `npm run desktop:dev`.
3. Run `npm run test:all` before opening a pull request.

Keep the kernel small, preserve review history and unknown content, and use the public extension SDK for optional capabilities. First-party and third-party extensions must use the same documented mechanisms.

## Pull requests

Keep changes focused, include tests for behavioral changes, document schema or SDK compatibility effects, and attach screenshots for visible UI work. Never commit signing credentials, user data, imported decks, or private crash reports.
