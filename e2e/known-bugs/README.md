# Known-bug reproducers

Place deterministic Playwright reproducers here only after filing a GitHub issue. Each test title must include the issue number. This suite is intentionally non-gating; once a defect is fixed, move its reproducer into the appropriate gating suite.

- [Issue #47](https://github.com/neoanki/neo-anki/issues/47): macOS signature and Gatekeeper acceptance for a downloaded published `.app`. Set `NEO_ANKI_KNOWN_BUG_MACOS_APP` on a macOS runner.
