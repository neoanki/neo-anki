Neo Anki 0.1.4 expands the public extension SDK so independently installed extensions can build complete provider-backed features without private APIs.

## Highlights

- Added host-mediated HTTPS requests restricted to the exact domains reviewed at extension installation.
- Added per-extension API-key storage encrypted through the operating system credential service.
- Added opaque, one-time capability handles claimed before third-party modules load.
- Settings panels now receive the read-only workspace and transactional command API.
- Review tools now receive read-only workspace media alongside the active item and card.
- Knowledge items support namespaced extension metadata for durable, syncable media relationships.
- Queued extension commands now commit against the latest workspace state, making chunked batch jobs safe.
- Install review shows every declared HTTPS destination.

These capabilities use the same public SDK and enforcement for every publisher. No first-party-only host API was introduced.

## Downloads

- macOS universal DMG and ZIP for Apple silicon and Intel
- Windows x64 installer
- Linux x64 AppImage

macOS and Windows packages are intentionally unsigned. Follow the included installation instructions when the operating system displays its security warning. Updates are installed manually.
