Neo Anki 0.1.5 hardens the public extension credential vault on Linux.

## Security fix

- Rejects Electron's `basic_text` Linux storage backend, which uses a hardcoded plaintext password and is not a secure operating-system credential store.
- Accepts Linux credential storage only when Secret Service or KWallet is available.
- Keeps the existing Keychain behavior on macOS and DPAPI behavior on Windows.

Extensions receive a clear error when secure credential storage is unavailable. No API keys are written in that state.

## Downloads

- macOS universal DMG and ZIP for Apple silicon and Intel
- Windows x64 installer
- Linux x64 AppImage

macOS and Windows packages are intentionally unsigned. Follow the included installation instructions when the operating system displays its security warning. Updates are installed manually.
