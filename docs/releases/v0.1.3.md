Neo Anki is a local-first spaced-repetition desktop app built around the time you actually have available.

## Highlights

- Set a sustainable daily time budget or start a short session whenever you have a few minutes.
- Let the planner fit due reviews and new material into today without creating an unmanageable future workload.
- Study different subjects in short, coherent context blocks instead of switching randomly on every card.
- Use FSRS scheduling, review priorities, learning goals, saved searches, learning packs, and collection insights.
- Create basic, reverse, cloze, typed-answer, image-occlusion, audio, and cited prompts.
- Import Anki packages and CSV files; export portable data and complete backups.
- Keep your workspace in local SQLite storage with verified media, recovery backups, and restart-safe persistence.
- Install bundled or independently published extensions through the same TypeScript SDK and lifecycle.

## Downloads

- macOS universal DMG and ZIP for Apple silicon and Intel
- Windows x64 installer
- Linux x64 AppImage

macOS and Windows packages are intentionally unsigned. Follow the included installation instructions when the operating system displays its security warning. Updates are installed manually.

Each platform includes installation guidance, SHA-256 checksums, a CycloneDX software bill of materials, and GitHub build-provenance attestations. Verify an artifact with:

```bash
gh attestation verify <artifact> --repo neoanki/neo-anki
```

## Extension trust

Installed extensions run as trusted code. Permission declarations communicate intended capabilities, but they are not a security sandbox. Review every package and install extensions only from publishers you trust.
