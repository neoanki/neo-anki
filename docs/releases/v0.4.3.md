# Neo Anki 0.4.3

Neo Anki 0.4.3 turns the new-user audit into enforceable product and release gates, with particular attention to workspace recovery, extension configuration, authoring continuity, and macOS trust.

## User-facing fixes

- A damaged workspace now enters an explicit recovery screen instead of silently substituting an older backup. Saving remains blocked until the user restores, exports, or starts empty, and the damaged source is preserved.
- Neo Anki keeps seven rolling workspace backups and opens the restore picker at the newest available backup.
- New Knowledge drafts survive extension installation and automatic renderer reloads.
- Built-in knowledge editing consistently uses Prompt, Answer, and optional Context labels; custom imported note models retain their named fields.
- Extension configuration frames can grow and shrink with their content, remain reachable on phone and tablet widths, and expose document language and titles to assistive technology.
- Dark-theme extension controls use a semantic foreground color with sufficient contrast.
- A focus-management defect that could redirect Answer or Context typing into Prompt has been fixed.

## Verification and safety

- Installed-artifact audits are explicit opt-in suites and are excluded from normal browser and desktop tests.
- Credential tests use a disposable secret protector only in non-packaged isolated test hosts. Packaged applications reject that backend, and developer-machine artifact tests do not write or decrypt Keychain credentials.
- Release acceptance now exercises the clean packaged journey, persistence, review editing/resume, restart, and durable workspace counts.
- macOS release publication requires Developer ID signing, Apple notarization, a stapled ticket, and successful Gatekeeper assessment. Unsigned macOS artifacts are no longer releasable.

## Companion extensions

Patch releases for Card Timer, Image Occlusion, Collection Insights, Anki & CSV Import/Export, Text to Speech, and Workspaces adopt the new semantic primary-button foreground. Publish them only after Neo Anki 0.4.3 is available.
