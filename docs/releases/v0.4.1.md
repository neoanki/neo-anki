# Neo Anki 0.4.1

Neo Anki 0.4.1 repairs the extension journey discovered during hands-on use of the 0.4 release. It is a desktop preview release; mobile-store binaries and hosted sync are not included.

## Extension installation

- Replaces the non-functional Review and Install step with a single Install or Update action.
- Reloads the desktop runtime automatically after extension changes and restores the active Extensions view, search, selection, and scroll position.
- Opens Import from Anki directly in the installed Interoperability surface, falling back to its marketplace listing only when it is not installed.

## Configuration

- Corrects the sandbox appearance contract so extension frames receive the app canvas, semantic surfaces, native dark controls, and live theme state.
- Sizes configuration frames to their content instead of reserving a large empty panel.
- Centralizes dark/light state and native control styling in the SDK while preserving the version-2 extension interface.
- Updates Interoperability and Text to Speech packages to remove hard-coded light colors and duplicate host-owned headings.

## Platform notes

- macOS and Windows packages remain unsigned. Each artifact includes checksums, a CycloneDX SBOM, installation guidance, and GitHub build-provenance attestations.
- Automatic application updates remain disabled. Upgrade through a verified release download or the NeoAnki Homebrew tap.

## Verification

The release pipeline validates linting, type safety, unit coverage, browser journeys, hidden-window desktop integration, mobile export, extension packaging, licensing, platform installers, checksums, SBOMs, and build provenance before creating the draft release.
