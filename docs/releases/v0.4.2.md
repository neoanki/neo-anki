# Neo Anki 0.4.2

Neo Anki 0.4.2 repairs the Text to Speech setup path from knowledge creation.

## Changes

- Unavailable authoring actions now explain the prerequisite and provide a direct setup action instead of presenting a dead disabled control.
- The New knowledge draft, including media and selected prompt types, remains intact while an extension is configured.
- Extensions opened from authoring provide a clear return to New knowledge.
- Knowledge fields remain populated when a post-save extension action fails, and retry targets the already-created item without creating a duplicate.
- The primary Add action is blocked while an extension retry is pending.

## Companion extension

Text to Speech 2.0.6 adds a focused OpenAI setup that stores the API key in device-encrypted storage and configures portable prompt audio in one action. System voices remain realtime-only and do not create audio files.
