# Neo Anki 0.5.0

Neo Anki 0.5.0 replaces executable extension settings frames with a host-rendered declarative configuration contract.

- Configure renders inert manifest controls for synchronized values and device-only credentials; it contains no extension iframe, worker call, command bridge, or network path.
- Settings support bounded fields, static options, two-level repeatable groups, safe JSON Pointer bindings, conditions, host validation, explicit Save/Discard, and dirty-navigation confirmation.
- Credentials expose status only and use explicit Set/Replace/Delete operations without loading existing plaintext into the renderer.
- SDK 2.3 removes the `settings` UI surface and `ui:settings` permission. Operational extension workflows remain on review, page, create, workspace, or migration surfaces.
- Card Timer 2.1.0 and Text to Speech 2.1.0 are the coordinated first-party cutover packages. TTS batch generation and provider voice discovery move to its dedicated tools page.

The core, both extension packages, and marketplace entries must be published together. Older settings-frame packages are intentionally unsupported; there is no legacy parser, compatibility adapter, or fallback iframe.
